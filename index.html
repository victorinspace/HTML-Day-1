Victor's Stylized GitHub Cheat Sheet

Hello there. Today we are going to go over some simple instructions for how to use GitHub!

These instructions are as told by Mike Sweeney of which the original text can be found here.

GitHub Directions:

  1. Step 1:
    • mkdir project-name
    • cd project-name
  2. Step 2:
    • git init
  3. Step 3:
    • go to github.com and create repo to match project name
    • and copy the ssh url
  4. Step 4:
    • git remote add origin "cmd-v"
  5. Step 5:
    • do stuff
  6. Step 6:
    • git add .
  7. Step 7:
    • git commit -m "commit message"
  8. Step 8:
    • git push origin master
  • Repeat steps 5 - 8
