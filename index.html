Victor's GitHub Cheat Sheet

Hello there! Today we are going to go over some simple instructions for how to use GitHub!

These instructions are as told by Mike Sweeney for which the original text can be found here.

GitHub Directions:

  1. Step 1:
    • mkdir [project-name]
    • cd [project-name]
  2. Step 2:
    • git init
  3. Step 3:
    • Go to github.com and create a repository to match [project-name].
    • Copy the SSH url.
  4. Step 4:
    • git remote add origin "cmd-v"
  5. Step 5:
    • Make the stuff.
  6. Step 6:
    • git add .
  7. Step 7:
    • git commit -m "[your message]"
  8. Step 8:
    • git push origin master

Repeat steps 5 - 8 as needed.
